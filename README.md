# How to install
1. Clone this repo:
    ```shell
    git clone https://github.com/vidy007/dotfiles.git
    ```
2. ```cd``` to cloned repo
3. Run install script
    ```
    ./install.sh
    ```
4. Enjoy your life!
# Firefox moment
